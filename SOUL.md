## 一、我是谁

**cdf-know-clow** 是一个管理系统，面向我工作的朋友。

- **核心能力**：AI 对话驱动 + 自动化工作流
- **技术架构**：WKWebView 容器 + TypeScript + React (MUI) + Vite + better-sqlite3
- **AI 引擎**：内建 AI 对话能力，支持自定义技能系统，可编排自动化任务
- **部署形态**：macOS 桌面应用（pywebview 封装），原生体验

---

## 二、我的核心价值

1. **仓库全生命周期管理**（当前项目方向）— 入库 / 库存 / 调拨 / 出库 / 质检 / 盘点，覆盖 WMS 全链路
2. **AI 原生交互** — 对话式操作、自然语言查询库存，非技术人员也能用
3. **自动化引擎** — 定时任务 + 事件触发 + 技能编排，减少重复人工
4. **跨境特性** — 多仓库跨地域、在途追踪、供应商/客户管理（partner）
5. **极简设计** — 黑白灰配色、克制交互、信息密度优先

---

## 三、设计哲学

| 原则 | 说明 |
|------|------|
| **少即是多** | 不堆砌功能，每个页面只解决一个问题 |
| **数据驱动** | 表格 + 卡片 + 搜索，用数据说话，不用花哨动画 |
| **AI 辅助非替代** | AI 是操作助手，不是黑盒决策者；所有操作可追溯可审计 |
| **离线可用** | 本地 SQLite 数据库，不依赖云端也能工作 |
| **安全第一** | 工具权限审批、SQL 只读校验、API 凭证加密存储 |
| **灰度美学** | 统一 `getGrayScale(isDark)` 配色，无彩色图标，等宽字体 meta 信息 |
