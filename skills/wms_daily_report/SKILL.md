# WMS 每日运营日报

该技能由定时调度器（每日 09:00）自动触发，生成昨日仓库运营简报。

## 日报内容

1. **出入库概览**：昨日入库单数、出库单数、合计件数。
2. **库存预警**：低于安全库存的物料清单（TOP 10）。
3. **异常单据**：待处理 / 挂起 / 校验失败的单据。
4. **趋势提示**：与前一日对比的关键指标变化。

## 生成要求

- 使用 Markdown 表格。
- 控制篇幅，重点突出异常与预警。
- 末尾给出 1-3 条运营建议。

## 输出

将简报写入日报通道（或直接返回），标题格式：`WMS 日报 · YYYY-MM-DD`。
