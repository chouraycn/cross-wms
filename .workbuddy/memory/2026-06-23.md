### 2. CDF Auto Model v2.0 智能路由引擎

**完全重写** `server/routes/modelSelector.ts`，删除旧版简化逻辑。

**5 大判定维度（加权打分 0~10 分）**:
| 维度 | 权重 | 评分逻辑 |
|------|------|----------|
| 媒体类型 | 10% | 图片/视频=10分（强制多模态），PDF=8分 |
| 上下文Token长度 | 30% | 占窗口>85%=10分，60~85%=7分，30~60%=4分 |
| 关键词意图 | 40%（最高） | 复杂关键词>=2个=9分，中等=5~6分，简单=2分 |
| 代码特征 | 20% | 代码块/编译报错/语言关键词累计评分 |
| 工具调用 | 额外加分 | MCP>=3或Skill>=2或工具调用>=5次，最高+2分 |

**4 层自动分流**:
| 层级 | 触发条件 | 目标模型标签 |
|------|----------|-------------|
| Vision 多模态 | media >= 8 | `multimodal`（排除 DeepSeek） |
| Tier3 强推理 | 总分 >= 6.5 | `reasoning` / `code` |
| Tier2 均衡 | 总分 >= 3.5 或 code >= 7 | `general` / `code` / `reasoning` |
| Tier1 轻量 | 默认 | `fast` / `costEffective` / `general` |

**新增 API**:
- `computeComplexityScore(input)` — 5维度评分引擎
- `selectFallbackModel(failedModelId, config, errorCategory)` — 故障降级
- `maybeUpgradeForToolUsage(...)` — Tool/MCP 联动升级
